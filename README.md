# React/Typescript Dashboard Template

![licence](https://img.shields.io/badge/licence-MIT-blue)

## Install Packages

```bash
  yarn install
```

## Run Locally

To run locally, run the following command

```bash
  yarn start
```

## Deployment

To create a production build

```bash
  yarn run build
```
